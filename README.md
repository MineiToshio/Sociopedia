## ðŸ›  Installation & Set Up

### Server

1. Go to the server folder
2. Install dependencies:

```sh
npm install
```

3. Set up env variables.

```
# .env
MONGO_URL=**********************
JWT_SECRET=**********************
PORT=**********************
```

3. Start server:

```sh
nodemon index.js
```

### Client

1. Go to the client folder
2. Install dependencies:

```sh
npm install
```

3. Start dev server:

```sh
npm start
```
